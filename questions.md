七维向量顺序与 `dimension.md` 一致：**视线 (gaze) → 审美 (filter) → 基调 (attitude) → 版图 (scope) → 声量 (voice) → 投入 (investment) → 边界 (combat)**。

**题号规范（Q1–Q25）**：每题题干必须以 **`Q` + 题号 + `.`** 开头（例如 **`Q1.`**、**`Q25.`**）。请勿只写 **`1.`**、**`2.`** —— 在 Markdown 里容易与有序列表语法混淆，也不便于和 `data/questions.ts` 里的 `id` 对照。下文 **Q1–Q25** 已按顺序编号。
